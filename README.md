Try Web etc.


# Setup on your local
Need Node.js and npm.

Get source code, install modules and start app.
```
$ git clone git@github.com:tayutaedomo/videojs-trial.git
$ cd videojs-trial
$ npm install
$ bin/www
```
Your app should now be running on http://localhost:3000.
